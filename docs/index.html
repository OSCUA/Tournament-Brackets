Key features

Multiple formats

Run Single Elimination, Double Elimination, Round Robin, or Swiss with configurable rounds.

Simple UI

Add players, generate rounds, record results, and view standings in one place.

Byes handled

Odd player counts? Byes are automatically assigned and scored correctly.

Supported formats

Single elimination

Players are eliminated after one loss. Progression continues until one champion remains.

Double elimination

Winners and losers brackets converge in a grand final, with reset support.

Round robin

Every player faces every other player. Standings by points, wins, and ties.

Swiss

Score-group pairings that avoid repeats, with configurable round count and bye fairness.

How to use

1. Install Python

Use Python 3.11 or 3.12 for best compatibility.

2. Clone the repository

git clone https://github.com/YourUser/YourRepo.git

3. Create a virtual environment

python -m venv venv && venv\Scripts\activate

4. Install dependencies

Choose a GUI flavor:

  • Tkinter: Built-in, no extra install.
  • Kivy (not tested): pip install kivy

5. Run the app

Pick the version you added (e.g., Tkinter or Kivy):

python main.py python kivy.py

If you prefer Kivy, use Python 3.11â€“3.12 and install:
